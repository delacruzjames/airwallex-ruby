# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-06-27

### Added

- Initial gem skeleton for `airwallex-ruby`
- Core module entry point with `Airwallex.configure` and `Airwallex.client`
- Minimal `Client`, `Configuration`, and `Error` classes
- Resource scaffolding: `Airwallex::Resources::BaseResource` and `Authentication`
- RSpec test suite for foundation classes
- RuboCop configuration
- Research notes in `docs/airwallex-research.md`
